# EdgeSense
| Edge-based Equipment Monitoring & Analytics Platform

## 1. 프로젝트 목적

반도체 설비의 진동 데이터를 수집하여, 이상 징후를 사전에 파악하고 조치를 취하기 위한 모니터링 시스템 개발을 목적으로 한다.

본 프로젝트를 통해 반도체 라인에서 장비 이상으로 발생하는 병목 원인을 미리 차단할 수 있다.

## 2. 프로젝트 범위

- Acceleration data acquisition
- Edge Computing
- Edge Communication
- RMS Analysis
- PSD Analysis
- Anomaly Detection
- Web DashBoard
